# ClearPath

ClearPath is a multilingual, voice-guided immigration navigation tool.
It helps users understand next steps, documents, and official resources
without providing legal advice.

## Tech Stack
- Next.js
- MongoDB Atlas
- OpenAI
- ElevenLabs
- Vercel

## Running Locally
1. Create `.env.local`
2. Run `npm install`
3. Run `npm run dev`

## Disclaimer
This app does not provide legal advice.
